bedroom is a latest version fabric base for minecraft clients. <br>

**plans for bedroom 1**
- events
- module system
- setting system
- command system
- ui stuff

**how to use** <br>
*for eclipse* <br>
- drag .zip release to a new folder
- open folder in cmd
- type gradlew eclipse
- type gradlew genSources
- boom... use
